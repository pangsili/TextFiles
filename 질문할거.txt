# 
일단 이 파일만의 특징을 찾으라 하셨는데 다른말로, 

어떠한 내용을 보고 여러 가지 중 이 파일이다 라는 걸 알 수 있는 내용을 찾아보라는 것과 같다고 생각합니다.

그런데 이 여러가지 파일이라는 범위가 지금 하고 있는 프로젝트 내에서인지,

아니면 세계의 파일들 중 이것만의 특징인지 궁금합니다.

# 
오늘 Level 2 를 수행하던 도중 들던 의문은 저 management 파일들과 같이 

동일한 부분이 매우 많은 것들이 아니라, UsageLocation 같은 프로그램은 

그 Level 1 에서 도출한 결과 자체가 해당 프로그램의 특징이 될 수 있지 않은가 궁금하다.


#
Get / Set-MsolUser Command 가 여러번 나와서 검색해보았는데, 

Azure Action Directory 에서 사용자를 불러온다, 바꾼다 라는 검색결과도 있었고, 


#
SPOST / SPOSTGP

SPOST, SPOSTGP 같은 뒤에 붙은거에 따로 뜻이 있는 것인가?

있다면 왜 검색해도 나오질 않는건가?


#
PhoneType Parameter

KRTeamExtensionNumber, TeamExtensionNumber 두 파일에서 차이점을 찾으려다가 발견한 유일한 차이로,

무엇을 의미하는지, 이 차이가 특징이 될 수 있는지가 궁금하다.

검색하기 위해서 PowerShell PhoneType Parameter 등 여러가지 해봤지만 만족스러운 답이 없었다.

파워쉘의 파라미터 들에 대해서 알고싶을때의 검색 방법도 알았으면 좋겠다.
