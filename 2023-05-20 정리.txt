무언가를 작업할때는 복사 / 붙여넣기를 이용하여 로컬 환경(내 컴퓨터의 바탕화면 등)에 가져와서

거기서 작업을 한 뒤, 본래의 위치에 다시 복붙 혹은 덮어쓰기로 넣는 것으로 작업해야 한다.

작업할 때 항상 백업파일을 만드는 습관도 만들자. 파일이름은 _bk 로 통일할 것.

또, 똑같은 커맨드를 다시 확인하는 경우에도, 해당 커맨드를 다시 손으로 복사/붙여넣기 하여서,

확실하게 확인하는 느리지만 신중한 방법이 일본에서의 기본 방식이다.


#
보통 여러가지 프로그램 (엑셀, winMarge, 한글 등)에서 ctrl + N 으로 새 창을 만들 수 있다. 

N은 New 에서 파생되었기 때문에, 웬만한 프로그램들에 적용되는 편이다.


# 
winMarge

사분(일본어) / 차분(한국어) : 차이점

ex) 차분 알아봐주세요 = 서로 다른점을 찾아봐주세요

F5로 새로고침, 재컴파일 할 수 있다. 하나 바꾸고 계속해서 확인해주는 것이 좋다.

완전히 같은 파일이라면 F5를 눌렀을 때, 같은 파일이라고 글자가 나오는데,

이것을 증거로 스크린샷을 떠놔서, 증거품으로 보여주는 방법도 좋다.

문자 수정하는 방법에는 

1. 복사 / 붙여넣기로 하는 방법, 2. Alt + 오른쪽 으로 하는 방법이 있는데, 

2번 방법은 해당 주황 줄을 모두 바꿔버리는 것이기 때문에, 잘 확인하고 사용하여야 한다.

우리는, 왼쪽에 웹파일, 오른쪽에 최신화 텍스트 파일을 놓고, 

웹파일에서 필수인데 텍스트 파일에서는 다르게 적용된 것들을 ex) 변수의 고정 값 등 

텍스트에 올바르게 바꿔주고, 텍스트가 변경된게 잘 적용되었는지 

확인 한 뒤에 텍스트를 복붙해서 다시 웹에다가 고쳐서 넣은다음 저장하였다.

엑셀

Ctrl + R
치환하는 기능으로, Alt + A 로 모두 치환 기능도 사용할 수 있다.

Ctrl + F
검색하는 기능인데, 시트별 / 파일별 로 나눌수도 있고, 해당 문서가 있는 폴더에 들어가서 

오른쪽 위에 검색어를 넣어서 해당 파일까지도 나올 수 있게 할 수 있다.

Alt + 드래그
블록지정으로 글자들을 선택 할 수가 있다.

중복제거
맨 위 작업표시줄? 에서 데이터 탭에서 중복제거를 할 수 있다.
